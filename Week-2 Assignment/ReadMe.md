## 🎯 Objective

The goal of this assignment is to:
- Understand prompt engineering concepts  
- Apply the RTCFR framework (Role, Task, Context, Format, Rules)  
- Design an intelligent AI assistant  
- Explore real-world applications in game development

## 🎮 Project Description

In this assignment, I designed an **AI Game Development Assistant** using prompt engineering techniques.

The system acts as a mentor that:
- Guides beginners in game development  
- Suggests ideas and mechanics  
- Provides step-by-step assistance  
- Helps users progress toward building a playable game
