# 🤖 Agent.AI – Advanced Generative AI Application

> 🚀 A secure, customizable, and interactive Gen AI platform built using Python and Streamlit.

---

## 📌 Overview

This project demonstrates the development of a **real-world Generative AI application** with authentication, API integration, and customizable AI interaction features.

Agent.AI is designed to simulate how modern AI platforms work, allowing users to securely log in, connect their API keys, and control how AI responses are generated.

---

## 🧠 Project Description

**Agent.AI** is an advanced Gen AI application that provides:

- Secure user authentication  
- API key-based AI access  
- Customizable response behavior  
- Interactive AI testing environment  

The system allows users to experiment with AI models in a controlled and personalized way.

---

## 🔐 Authentication System

The application includes a secure login system:

- Email-based user login  
- OTP (One-Time Password) verification  
- OTP sent via email using SMTP  
- Access granted only after verification  

👉 Ensures **user security and controlled access**

---

## 🔑 API Key Integration

Users can:

- Enter their own API key  
- Connect to AI services  
- Test different models  
- Maintain privacy (no hardcoded keys)

👉 Enables **flexible and personalized AI usage**

---

## ⚙️ Customization Features (Core Highlight 🔥)
