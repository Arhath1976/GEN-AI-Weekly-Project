# 🚀 Gen AI API Tester App

A fully functional **Generative AI testing platform** built using **Python and Streamlit**, developed as part of a Week 4 assignment.

This application allows users to securely log in, connect their own API keys, automatically detect AI models, and interact with them efficiently while optimizing token usage.

---

## 🌟 Project Overview

This project focuses on building a real-world **Gen AI application** that demonstrates:

- Secure user authentication
- API integration with user-provided keys
- Intelligent model detection
- Token-efficient response handling
- Interactive UI for prompt testing

---

## ✨ Key Features

### 🔐 Authentication System
- User login system  
- New account creation  
- Secure session handling  

### 🔑 API Key Integration
- Users input their own API keys  
- Keys are used dynamically per session  

### 🧠 Automatic Model Detection
- The app intelligently detects the model linked to the API key  
- Removes manual configuration for users  

### 💬 AI Interaction Interface
- Users can input prompts/questions  
- Receive real-time AI-generated responses  

### 🎛️ Response Control
- Adjustable response length  
- Helps control token usage  

### 💰 Token Saving Mode
- Optimized responses to reduce API cost  
- Ideal for efficient usage  

---

## 🛠️ Tech Stack

- **Frontend/UI:** Streamlit  
- **Backend:** Python  
- **AI Integration:** API-based (user-provided keys)  
- **Authentication:** Custom login & account system
